Etiquetas < input > y sus variantes

Tipo texto "text" a

  • crea campo texto en una linea
  • acxepta cualquier tipo de texto alfanumerico

Tipo "password"

  • como el campo de texto pero los caracteres se ocultan

Tipo "email"

  • Acepta y valida una direccion de correo:

Tipo "number"

  • Solo acepta numeros y puede incluir atributos como min, max, step.

Tipo "date"

  • proporciona un selector de fecha

Tipo "checkbox"

  • crea una casilla de verificacion que permita al usuario seleccionar una varias opciones

Tipo "radio"

  • crea una casilla de verificacion que permita al usuario seleccionar una varias opciones

Tipo "file"

  • permite cargar un archivo

Tipo "submit"

  • permite enviar formulario

Tipo "reset"

  • Boton de reinicio que restaura los valores predeterminado

Tipo "hidden"

  • crea un campo oculto que no es visible para el usuario, pero si puede almacenar datos
